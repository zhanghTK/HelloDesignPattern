# 单例

## 实现方式

|    实现     | 线程安全 | 调用效率 | 延时加载 |        其他         |
| :-------: | :--: | :--: | :--: | :---------------: |
|   饿汉单例    |  安全  |  高   |  否   |                   |
|  简单懒汉单例   | 不安全  |  高   |  是   |                   |
|  加锁懒汉单例   |  安全  |  低   |  是   |                   |
| 静态内部类懒汉单例 |  安全  |  高   |  是   |    依赖静态内部类初始化     |
|   双重校验锁   |  安全  |  较高  |  是   | JVM底层内部模型原因可能出现问题 |
|   枚举单例    |  安全  |  高   |  否   |  可以避免反射和反序列化生成对象  |
