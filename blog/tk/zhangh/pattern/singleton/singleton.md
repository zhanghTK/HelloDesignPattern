# 单例

## 五种实现方式

- 饿汉单例


- 懒汉单例
- 双重校验锁
- 静态内部类
- 枚举单例

| 实现        | 线程安全 | 调用效率 | 延时加载 | 其他          |
| --------- | ---- | ---- | ---- | ----------- |
| 饿汉单例      | 安全   |      | 否    |             |
| 简单懒汉单例    | 不安全  |      | 是    |             |
| 加锁懒汉单例    | 安全   | 低    | 是    |             |
| 静态内部类懒汉单例 | 安全   | 高    | 是    | 依赖静态内部类初始化  |
| 双重校验锁     | 安全   | 较高   | 是    | 依赖JVM，可能有问题 |
